pReProx

A Reverse-Proxy Service using netcat, bash and PHP.

Installation

Compile newredir.c to newredir and set the owner/group to root:root. Then set the 'set uid' flag (chmod u+s newredir). Edit config.php to your linking (set SECRET to something random), then log into the admin panel and add / remove the ports you want to use. Also make sure the firewall forwards all used ports.

A setup script for instantserver.io servers can be found here
